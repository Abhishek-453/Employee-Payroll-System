# Employee Payroll System 💼

A console-based Java application designed to manage employee data and automate payroll calculations using **Object-Oriented Programming (OOP)** principles.

## 🚀 Key Features
* **Dynamic Management:** Add and remove employees from the system.
* **Automated Calculations:** Monthly salaries for full-time and hourly wages for part-time staff.
* **Polymorphic Design:** Handles different employee types seamlessly.

## 🧠 OOP Concepts Implemented
* **Abstraction:** Abstract class `Employee` defines the core structure.
* **Inheritance:** `FullTimeEmployee` and `PartTimeEmployee` extend the base class.
* **Polymorphism:** Method Overriding for `calculateSalary()`.
* **Encapsulation:** Private fields with public getters and constructors.

## 🛠️ Technology Stack
* **Language:** Java (JDK)
* **IDE:** IntelliJ IDEA
